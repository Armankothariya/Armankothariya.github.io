## 🖤 About The Portfolio

This is my personal portfolio and digital hub, built to reflect my core philosophy: **the mind as a system to be mapped**.

It goes beyond a simple resume, presenting my work at the intersection of:

* 🧠 **Brain-Computer Interfaces (BCI)** & Prosthetics
* 🤖 **Artificial Intelligence & Multimodal Systems**
* 🌍 **Citizen Science (Asteroid Detection)**

The design emphasizes clarity, signal over noise, and a systems-thinking aesthetic.

> *"The mind is not a mystery to be feared — but a system to be mapped."* — Arman
